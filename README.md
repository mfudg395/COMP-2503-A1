# COMP-2503-A1
Assignment 1 for COMP-2503 at Mount Royal University.

This is a simple program that reads the frequency of words from a file and prints them to the system output stream. The ten most frequently
and least frequency words will be listed, and a list of all words in alphapetical order will be printed at the end. Words apart of a given
"stop" list are not included in counts.
